This repository contains the official training material, lab exercises, and final projects for API Testing delivered during my time as a Mentor/Instructor. These materials were extensively used at companies like **DOKU** to transition manual QA engineers into Automation Engineers.

## 🎯 Objective
To provide a structured, step-by-step curriculum that breaks down API testing from "what is an HTTP request?" to implementing a complete programmatic API automation test suite.

## 📚 Curriculum Breakdown
1. **Module 1**: Understanding REST APIs (HTTP methods, Status Codes, Headers)
2. **Module 2**: Manual API Testing (Using Postman / cURL)
3. **Module 3**: Introduction to Automation Frameworks
4. **Module 4**: Parsing JSON Responses & Complex Assertions
5. **Final Project**: Implementing a complete CRUD cycle test suite against a mock application.

## 🎓 Mentorship Philosophy
I believe that test automation is 80% engineering mindset and 20% tool syntax. This repository focuses heavily on the *why* alongside the *how*, ensuring that learners can adapt to any testing tool (Pytest, Katalon, RestAssured) by mastering fundamental API principles.
